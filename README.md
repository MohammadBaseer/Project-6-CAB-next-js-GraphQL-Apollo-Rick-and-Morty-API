# Next.js 14 App - Rick and Morty API Project

## Features

- 🧑‍🚀 **Display Characters from Rick and Morty API**
- 💻 **Client-Side Rendering**
- 🌐 **Server-Side Rendering**
- 🚀 **Server-Side GraphQL for Displaying Episodes**
